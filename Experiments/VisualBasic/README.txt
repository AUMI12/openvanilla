== How to Build [=Experiment/Visual Basic] ==

Preparation Work

0 check out the path [=trunk/Experiments/VisualBasic] (it's self sufficient, no other dependencies needed)
0 *copy everything, including the sub-diretories, to [=C:\OVVBPOJ], make sure the case is right*
0 before you go, check if [=C:\OVVBPOJ\OpenVanilla] and [=C:\OVVBPOJ\OVIMPOJ-Holo] exist and have files within it

Build

0 click "Start Menu" and go to "MS VC++ Toolkit", there is a Command line environment menu item, click on it
0 you'll enter Windows's command line console, with all VC++ environment variables properly set
0 go to [=C:\OVVBPOJ]
0 type [=make.bat]
0 after the build process, an [=OVIMPOJ.DLL] should be there

Importing Forms and Modules in Visual Basic

0 進 MS Word -> 工具 -> 巨集 -> Visual Basic 編輯器
0 找到 Normal.dot
0 在上面點右鍵，選匯入
0 把 C:\OVVBPOJ\POJForm.frm , .frx, .bas 三個檔案匯入
0 離開 Visual Basic 編輯器
0 選工具 -> 巨集... -> 然後應該會看到巨集清單
0 選那個 ShowPOJ 就行了
