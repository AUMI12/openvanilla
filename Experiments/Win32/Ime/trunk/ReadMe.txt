========================================================================
    動態連結程式庫 : OVIME 專案概觀
========================================================================
這個檔案含有構成 OVIME 應用程式之每一個檔案的內容摘要。


OVIME.vcproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔案。
    它含有產生該檔案之 Visual C++  的版本資訊，以及有關使用應用程式精靈所選取之平台、組態和專案功能的資訊。

OVIME.cpp
    這是主要 DLL 原始程式檔。

	這個 DLL 建立之後不會匯出任何符號。
	因此，在建置它的時候不會產生任何 .lib 檔案。
	如果您希望這個專案成為其他專案的相依專案，您必須加入程式碼從 DLL 匯出某些符號以便產生匯出程式庫，
	或者，您可以在專案的 [屬性頁] 對話方塊中，將 [連結器] 資料夾 [一般] 屬性頁上的 [忽略輸入程式庫] 屬性設定為 [是]。

安裝:

請使用本目錄下的 IMEInstaller 來安裝 IME/DLL 檔案
