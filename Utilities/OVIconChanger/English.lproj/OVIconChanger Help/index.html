OpenVanilla Icon Changer is an utility which allow you to customize the icon displayed in the input menu of OpenVanilla. It provides not only varies of built-in icon files, an easy way to replace the icon, but also a conversion tool to covert your photos or drawings into the file format of an icon of Mac OS X.

Change Icon

When you launch OpenVanilla Icon Changer, you will see that there is a table which contains icons. Choose a row from the table by click one of the icons, then click the "Change Icon" button in the bottom. Then, OpenVanilla Icon Changer will ask you to logout and re-login. When you login again, you will see the change has happened: you can see the icon of OpenVanilla in the input method menu and the International System Preference Pane becomes to the icon that you chosen.

Add a New Icon to the List

You can create your own icons by tools such as Apple Computer's Icon Composer bundled in XCode development tool, or any other tools by Apple's 3rd Party. The icon should be in ICNS format and contains a 16x16 image. Although ICNS file format can support high solution images, however, the Apple menu bar can only display a 16x16 image for an Input Method.

When you complete designing an icon, launch OpenVanilla Icon Changer, and choose "Add New Customized Icon" from the "File" menu, a dialog window will appear. Please choose your icon file in this dialog window by browsing files and folders in your disks, click "Open", OpenVanilla Icon Changer will copy your file to proper folder and display it in the table.

Convert an Image as an Icon

If you have no idea about how to create an icon with any design tools, we provide an easy way to let you obtain your own icons. Choose "Convert an Image to an Icon" command in the "File" menu, select an image file, OpenVanilla Icon Changer will convert it to as icon and add it to the icon table. The source image could be a jpeg, bmp, gif, png or tiff file, or in other popular image format.
