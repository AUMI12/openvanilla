OpenVanilla 圖示修改器是一套可以讓您自行設計、修改在輸入法選單上索顯示的 OpenVanilla 圖示的工具。這套工具提供了一套快速更換圖示的方式，也提供一些內建的圖示供您更換，另外，還內建一套將圖片檔案轉為圖示格式的轉換工具，讓您輕鬆建立可以供 Mac OS X 使用的圖示。

修改圖示

在啟動 OpenVanilla 圖示修改器之後，您可以在主視窗中看到一個包含許多圖示的列表，您只要選擇列表中的圖示，然後按下畫面下方的「修改圖示」按鈕，便可以將 OpenVanilla 的圖示換成您所指定的圖示。不過，在修改之後，您需要先登出，然後重新登入，才可以看到變更圖示的效果。在變更之後，無論是輸入法選單，或是「國際設定」上，都可以看到 OpenVanilla 已經換成您的圖示了。

新增圖示

您可以使用像蘋果在 XCode 開發工具中內建的 Icon Composer 等工具，或是其他蘋果的第三方協力廠商開發的工具，設計屬於您自己的圖示。圖示檔案必須是 ICNS 格式，檔案中包含一個 16x16 大小的圖片。雖然 ICNS 格式可以讓您設計高解析度的圖示，不過，畢竟在蘋果選單上要顯示一套輸入法，只能夠顯示 16x16 的圖示，如果圖片大小超過這個範圍，會造成顯示上的問題。

當您完成圖示設計，請啟動 OpenVanilla 圖示修改器，然後從「檔案」選單上選擇「新增圖示到圖示清單指令」，便會跳出一個對話視窗，請在對話視窗中指定您設計的檔案，選擇「開啟」， OpenVanilla 圖示修改器就會幫您把檔案複製到正確的目錄，並且加入到圖示列表中。

將影像檔案轉成圖示

如果您不清楚如何使用上述工具設計圖示，我們也提供一套簡單產生圖示的方法。請從「檔案」選單上選擇「將圖片轉換成圖示」指令，之後，請選擇任何一張圖片檔案，系統就會自動將圖片轉換成圖示並加入到圖示清單中。您可以使用 jpeg、bmp、gif、png、tiff等常用格式的圖檔，轉換成圖示。
