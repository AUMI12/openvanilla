Introduction to the Input Method Table of OpenVanilla

One of the Input Method Modules of OpenVanilla, the Generic Input Method Module, was designed for the risk to load all of the Table-based Input Methods. You can find these tables of such Input Methods in the "OVIMGeneric" folder which locates in "/Library/OpenVanilla/[Version]/Modules". The tables are stored in plain text files with file extension "cin".

Because they are plain text files, you can use any text editors such as the Built-in TextEdit of Mac OS X, TextMate, Smultron, SubEthaEdit, Vim, Emacs and so on. However, to edit these tables, using the Generic Input Method Editor that you are now using, is the most convenient way. Remenber, when you edit a table file with the text editors other than the OpenVanilla Generic Input Method Table Editor, please leave the files in UTF-8 text encoding. And if you want to make these tables in effect, you must put these files into the folder mentioned.

If you want to disable a Table-based input method, you can just delete the file, or you can use the OpenVanilla Preference Utility to disable them, let them not to be displayed in the menu of OpenVanilla.
