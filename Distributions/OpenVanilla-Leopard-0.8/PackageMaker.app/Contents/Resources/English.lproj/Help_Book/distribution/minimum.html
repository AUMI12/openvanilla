A Minimal Distribution

To create a minimal, functional distribution, the following must be done:

  1. Create the packages to be bundled.
    More Info…
  2. Give the distribution a title in the “Installer Interface” tab.
    More Info…
  3. Create choices for the items you want to install. Multiple packages can be attached to a single choice.
    More Info…
  4. Add the packages to the appropriate choices.
    More Info…
  5. Make sure the information for each package is correct.
    More Info…
  6. Build the distribution using the menu item Project > Build…
