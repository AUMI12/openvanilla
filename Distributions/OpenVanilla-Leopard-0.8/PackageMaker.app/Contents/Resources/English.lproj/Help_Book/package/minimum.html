A Minimal Package

To create a minimal, functional package, the following must be done:

  1. Give your package a title in the “Installer Interface” tab.
    More Info…
  2. Specify the files to install in the “Contents Tab”.
    More Info…
  3. Verify that the file permissions are correct in the “Contents Tab”.
    More Info…
  4. Specify a default install location in the “Configuration Tab”.
    More Info…
  5. Specify an authorization level in the “Configuration Tab”.
    More Info…
  6. Provide a unique bundle identifier and version for your package in the “Package Version” tab.
    More Info…
  7. Build the package using Project > Build… menu item.
