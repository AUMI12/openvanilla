The Resources Pane

This pane prompts you for interface resources of your package, metapackage, or distribution: the welcome, readme, license, conclusion, and background image files. The assistant currently provides no mechanism to localize these files. For more information on these resources, see the discussion in the Packages section.

Files can be specified by drag-n-drop or by using the “Choose…” buttons. No files are required.
