Package Setup Assistant

The Package Setup Assistant is a different interface for package creation. It is accessible from the new file panel (File > New), “Assist Me…” button. It is capable of creating packages, metapackages, or distribution based on what settings are specified and what items are specified for the payload.

The assistant is intended to allow for quick and easy package creation with intelligent defaults. If you need different custom settings, simply open the output .pkg or .mpkg in PackageMaker.

The assistant will walk you through the package creation process in the following steps:

  1. Welcome: Prompts for basic package information.

  2. Contents: Prompts for the items to install.
    More Info…
  3. Bundle Identifiers: Displays what packages will be created and prompts for bundle identifiers and versions.
    More Info…
  4. Metadata: Prompts for package metadata. Not displayed when creating a distribution.
    More Info…
  5. Resources: Prompts for resources (e.g. license, readme, etc.)

  6. Package Creation: Will create all packages and bundle them in a metapackage or distribution if necessary.
