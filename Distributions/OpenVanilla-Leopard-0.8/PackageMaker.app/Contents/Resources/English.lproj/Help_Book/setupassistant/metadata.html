The Package Metadata Pane

This pane will only be shown when creating a package or metapackage, as it is not applicable for distributions. It prompts for some metadata information to associate with the package or metapackage.

For a package, it will prompt for a description and a post-install action. Both are optional.

For a metapackage, it will only prompt for a description.
