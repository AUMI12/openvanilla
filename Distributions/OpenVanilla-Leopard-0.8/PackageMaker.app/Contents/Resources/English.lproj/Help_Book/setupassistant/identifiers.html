The Bundle Identifiers Pane

This pane prompts you for the bundle identifiers and version number for each package that will be created. It provides default bundle identifiers based on the title and organization provided in the Welcome pane and the component titles for the items specified in the Contents pane. As always, each bundle identifier should be unique.

The assistant will also provide a default version number for each item. If the item is a bundle, this number will be the version number found in the bundle. If the item is not a bundle or no version number can be found, it will default to 1.0.

For more information on bundle identifiers and versions, see the discussion in the Package section.
