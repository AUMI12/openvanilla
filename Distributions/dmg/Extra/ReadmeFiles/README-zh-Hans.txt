OpenVanilla 0.7.2 for OS X
选用模块及输入法资料表套件

Copyright (c) 2004-2006 The OpenVanilla Project

自OpenVanilla 0.7.2开始，额外安装的模块和输入法资料表，可直接安装在使用者
家目录下的一个特定位置。以下便说明模块及资料表的安装方法，并针对本套件内容
做一简单描述。


== 安装模块的方法 ==

就拿「Big5内码输入法」这个模块为例吧，目录内有一个名叫「OVIMBig5.dylib」
档案，这是安装这套输入法唯一需要的档案。安装方法很简单，只要将它拷贝至以下
目录：

    ~/Library/OpenVanilla/0.7.2/Modules/

就可以了。上面的「~」符号代表的是您的家目录所在。如果您用的是Finder，您会
在您家目录下的「资源库」里找到OpenVanilla目录，再一层层往下找，就会看到
0.7.2目录下面的Modules目录了。

注意：请不要把「模块─Big5内码输入法」这个目录给直接拷贝进去，请拷贝目录
的内容物。

如果您用过OpenVanilla 0.7.2之前的版本，您可能知道，以前的模块必须安装在
主磁碟机下的「资源库」里面的那个OpenVanilla目录：

    /Library/OpenVanilla/version_number/Modules
    
如今已经没必要这么做了。不过如果您把档案拷到该目录，还是一样会生效的。如此
则使用您同一台机器的所有使用者都可以用到该模块。

有一些模块，像「注音（大字集）」模块，里面除了.dylib外，还有一个资料目录。
例如同一个例子里，除了有OVIMPhoneticSQLite.dylib外，还有一个同样名叫
OVIMPhoneticSQLite的目录。请您安装时也把目录拷贝过去。简言之：只要拷贝
.dylib及其资料目录（如果有的话）进去上述的 Modules/ 目录就可以用了。

最后要说明的是：在您安装之后，您可能发现原来已经开启的应用程序，并无法使用
这些模块。反而是新开起来的应用程序，可以使用新安装的模块。这是由于OS X的
特殊输入法架构造成的副作用。


== 资料表的安装方法 ==

以 .cin 结尾的资料表，要不就是给「泛用输入法产生器模块」（OVIMGeneric）
使用，要不就是给「字根反查输出模块」（OVOFReverseLookup）所用。要分辨的方
法也很简单，后者通常命名中有 lookup- 这个字首。

要安装 .cin 的输入法资料表很简单，只要将 .cin 档拷至以下目录就可以了：

    ~/Library/OpenVanilla/0.7.2/UserSpace/OVIMGeneric/

至于反查表则是复制到下列目录：    

    ~/Library/OpenVanilla/0.7.2/UserSpace/OVOFReverseLookup/

安装后的情况，就跟上述的模块档案一样。


== 套件内容简介：输入法模块部份 =

* Big5内码输入法：输入 Big5 内码输出中文字

* 注音（大字集）：提供将近 58,000 条注音字目，但需注意的是，其中有半数
  以上仅有字码存在，而无现有 Unicode 字型支援。但有了这个模块就可以打出
  「瀞」、「堃」、「喆」等字了。需注意的是，由于设计上是要与「传统注音」
  模块分立并存的，该模块的键盘排列必须从「偏好设定」的「个别模块设定」tab
  中的「其他模块的设定」中，动到keyboardLayout这个选项才能改变。其中
  0是内定值的传统排列，1是倚天排列。

* 英数：对不想离开OpenVanilla或是熟悉OpenVanilla快速键设定的朋友，
  这个「输入法」可能很有帮助。提供除了按CAPS LOCK以外的另一种选择。

* 藏文：zonble的藏文输入法，详细的使用说明及发展缘由，可参考zonble的
  blog: http://zonble.twbbs.org/archives/2004_12/669.php

* UIM-Anthy 日文平假名模式：这是个相当完整的日文输入法，可以选字选词，
  使用的是UIM这套著名的输入法引擎里提供的Anthy输入法。这个模块无法独立
  使用，必须先行安装MacUIM (http://sourceforge.jp/projects/macuim/)，
  请安装 0.4.1-1 版。有了MacUIM后，这个模块便可以在OpenVanilla内使用。
  功能及美观上不能跟MacUIM比，但对仅需偶尔输入日文的人，可能会有用。

* Unicode内码输入法：和Big5内码输入法类似。打入内码，出现Unicode字。


== 套件内容简介：文字处理模块部份 ==

* 注音文防治滤嘴：把这个 output filter 装起来并打开，就打不出注音文了。

* 英法德智慧引号：在英数字模式下（例如使用上述的「英数」输入法，或是在酷音
  或其他输入法里按CAPS LOCK进入英数字模式），可以作到类似Word的智慧引号
  功能，有英语、法语、德语三种不同的单双引号模式。
  
* 自动输出HTML Entities：打 & 出 &amp; ，打 < 出 &lt; ，对一些硬派
  手工HTML流的网页设计者可能会很有用˙。
  
* 显示输出字之Unicode内码：打完字，显示字码及 UTF-8 字码，包括范围超过
  U+FFFF 的字码也能显示， 如果是后者除了显示 Unicode 字码，还会显示
  UTF-16 surrogate。

* 将「．」符号转换为中文版Wikipedia格式：中文版Wikipedia用的「．」符号
  字码跟一般中文输入法所出的字码不同，有了这个 output filter 就可以不用
  一直查表了。
  

== 套件内容简介：输入法资料表格部份 ==

* 仓颉（大字集）、简易（大字集）：和大字集注音类似，但这两种输入法只需
  资料表便可以使用，不像大字集注音需另外编制模块才能拿来用。

* 仓颉-J、仓颉五代、顺序速成、快仓七代：由crossbone提供的各种仓颉版本，
  都具有香港外字支援。

* 四角号码、电报码、粤语拼音：一些由 Xcin 计划的资料表所转换来的输入法。

* 吴语（全字集）、吴语（繁体）、吴语（简体）：吴语输入法资料表格，
  来源为 http://input.foruto.com/wu/download.html
  
* 简体双拼：简体中文的双拼输入法，同样源自 Xcin 计划。
