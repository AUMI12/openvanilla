OpenVanilla 0.7.2 for OS X
選用模組及輸入法資料表套件

Copyright (c) 2004-2006 The OpenVanilla Project

自OpenVanilla 0.7.2開始，額外安裝的模組和輸入法資料表，可直接安裝在使用者
家目錄下的一個特定位置。以下便說明模組及資料表的安裝方法，並針對本套件內容
做一簡單描述。


== 安裝模組的方法 ==

就拿「Big5內碼輸入法」這個模組為例吧，目錄內有一個名叫「OVIMBig5.dylib」
檔案，這是安裝這套輸入法唯一需要的檔案。安裝方法很簡單，只要將它拷貝至以下
目錄：

    ~/Library/OpenVanilla/0.7.2/Modules/

就可以了。上面的「~」符號代表的是您的家目錄所在。如果您用的是Finder，您會
在您家目錄下的「資源庫」裡找到OpenVanilla目錄，再一層層往下找，就會看到
0.7.2目錄下面的Modules目錄了。

注意：請不要把「模組─Big5內碼輸入法」這個目錄給直接拷貝進去，請拷貝目錄
的內容物。

如果您用過OpenVanilla 0.7.2之前的版本，您可能知道，以前的模組必須安裝在
主磁碟機下的「資源庫」裡面的那個OpenVanilla目錄：

    /Library/OpenVanilla/version_number/Modules
    
如今已經沒必要這麼做了。不過如果您把檔案拷到該目錄，還是一樣會生效的。如此
則使用您同一台機器的所有使用者都可以用到該模組。

有一些模組，像「注音（大字集）」模組，裡面除了.dylib外，還有一個資料目錄。
例如同一個例子裡，除了有OVIMPhoneticSQLite.dylib外，還有一個同樣名叫
OVIMPhoneticSQLite的目錄。請您安裝時也把目錄拷貝過去。簡言之：只要拷貝
.dylib及其資料目錄（如果有的話）進去上述的 Modules/ 目錄就可以用了。

最後要說明的是：在您安裝之後，您可能發現原來已經開啟的應用程式，並無法使用
這些模組。反而是新開起來的應用程式，可以使用新安裝的模組。這是由於OS X的
特殊輸入法架構造成的副作用。


== 資料表的安裝方法 ==

以 .cin 結尾的資料表，要不就是給「泛用輸入法產生器模組」（OVIMGeneric）
使用，要不就是給「字根反查輸出模組」（OVOFReverseLookup）所用。要分辨的方
法也很簡單，後者通常命名中有 lookup- 這個字首。

要安裝 .cin 的輸入法資料表很簡單，只要將 .cin 檔拷至以下目錄就可以了：

    ~/Library/OpenVanilla/0.7.2/UserSpace/OVIMGeneric/

至於反查表則是複製到下列目錄：    

    ~/Library/OpenVanilla/0.7.2/UserSpace/OVOFReverseLookup/

安裝後的情況，就跟上述的模組檔案一樣。


== 套件內容簡介：輸入法模組部份 ==

* Big5內碼輸入法：輸入 Big5 內碼輸出中文字

* 注音（大字集）：提供將近 58,000 條注音字目，但需注意的是，其中有半數
  以上僅有字碼存在，而無現有 Unicode 字型支援。但有了這個模組就可以打出
  「瀞」、「堃」、「喆」等字了。需注意的是，由於設計上是要與「傳統注音」
  模組分立併存的，該模組的鍵盤排列必須從「偏好設定」的「個別模組設定」tab
  中的「其他模組的設定」中，動到keyboardLayout這個選項才能改變。其中
  0是內定值的傳統排列，1是倚天排列。

* 英數：對不想離開OpenVanilla或是熟悉OpenVanilla快速鍵設定的朋友，
  這個「輸入法」可能很有幫助。提供除了按CAPS LOCK以外的另一種選擇。

* 藏文：zonble的藏文輸入法，詳細的使用說明及發展緣由，可參考zonble的
  blog: http://zonble.twbbs.org/archives/2004_12/669.php

* UIM-Anthy 日文平假名模式：這是個相當完整的日文輸入法，可以選字選詞，
  使用的是UIM這套著名的輸入法引擎裡提供的Anthy輸入法。這個模組無法獨立
  使用，必須先行安裝MacUIM (http://sourceforge.jp/projects/macuim/)，
  請安裝 0.4.1-1 版。有了MacUIM後，這個模組便可以在OpenVanilla內使用。
  功能及美觀上不能跟MacUIM比，但對僅需偶爾輸入日文的人，可能會有用。

* Unicode內碼輸入法：和Big5內碼輸入法類似。打入內碼，出現Unicode字。


== 套件內容簡介：文字處理模組部份 ==

* 注音文防治濾嘴：把這個 output filter 裝起來並打開，就打不出注音文了。

* 英法德智慧引號：在英數字模式下（例如使用上述的「英數」輸入法，或是在酷音
  或其他輸入法裡按CAPS LOCK進入英數字模式），可以作到類似Word的智慧引號
  功能，有英語、法語、德語三種不同的單雙引號模式。
  
* 自動輸出HTML Entities：打 & 出 &amp; ，打 < 出 &lt; ，對一些硬派
  手工HTML流的網頁設計者可能會很有用˙。
  
* 顯示輸出字之Unicode內碼：打完字，顯示字碼及 UTF-8 字碼，包括範圍超過
  U+FFFF 的字碼也能顯示， 如果是後者除了顯示 Unicode 字碼，還會顯示
  UTF-16 surrogate。

* 將「．」符號轉換為中文版Wikipedia格式：中文版Wikipedia用的「．」符號
  字碼跟一般中文輸入法所出的字碼不同，有了這個 output filter 就可以不用
  一直查表了。
  

== 套件內容簡介：輸入法資料表格部份 ==

* 倉頡（大字集）、簡易（大字集）：和大字集注音類似，但這兩種輸入法只需
  資料表便可以使用，不像大字集注音需另外編製模組才能拿來用。

* 倉頡-J、倉頡五代、順序速成、快倉七代：由crossbone提供的各種倉頡版本，
  都具有香港外字支援。

* 四角號碼、電報碼、粵語拼音：一些由 Xcin 計劃的資料表所轉換來的輸入法。

* 吳語（全字集）、吳語（繁體）、吳語（簡體）：吳語輸入法資料表格，
  來源為 http://input.foruto.com/wu/download.html
  
* 簡體雙拼：簡體中文的雙拼輸入法，同樣源自 Xcin 計劃。
